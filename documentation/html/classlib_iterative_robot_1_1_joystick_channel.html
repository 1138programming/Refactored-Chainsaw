Detailed Description

The JoystickChannel class is used to run and stop Commands and CommandGroups using a joystick channel. It implements the Trigger class and inherits all of its events. The events associated with it are: when the channel passes a threshold forwards, while the channel is past the threshold, when the channel passes the threshold backwards, and while the button is within the threshold.
