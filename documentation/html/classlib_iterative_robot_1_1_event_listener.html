Detailed Description

The EventListener class is the base class for event listeners such as the Trigger class. When an addEventListener is instantiated, it is automatically added to the EventScheduler, which calls its checkConditions method repeatedly. This is used to run or stop Commands or CommandGroups when certain conditions are met.
